# EYT Pro X Wireless Headset Website

## Deploy to Vercel in 3 steps:

### 1. Upload to GitHub
- Go to github.com → New repository → name it `eyt-headset`
- Upload ALL files from this zip (keep folder structure)
- Click "Commit changes"

### 2. Deploy on Vercel
- Go to vercel.com → Sign up free with GitHub
- Click "Add New Project" → select your `eyt-headset` repo
- Keep all defaults → click **Deploy**
- Live in ~60 seconds at `eyt-headset.vercel.app`

### 3. Get on Google
- Go to search.google.com/search-console
- Add your Vercel URL → Request Indexing

## Run locally:
```bash
npm install
npm run dev
```
Open http://localhost:3000
